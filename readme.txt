This plugin redirects WordPress Sign-up to homepage if is activated "Hide Backend" module in iThemes Security settings.

== Description ==

This is a simple plugin that redirects WordPress Sign-up to homepage if is activated "Hide Backend" module in iThemes Security settings.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/redirect-wp-signup-for-itsec` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
